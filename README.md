# 2.気象データ解析プログラム
| ソース名 | 概要 |
|---|---|
| temp_trend_anal.py| 26年分の日ごとの平均気温データから26年間の平均気温のトレンドを示すプログラム |
| climate_trend_analysis.py| アップデート版 |
## 主なデータ処理
- 温度の季節要因をFFTを使ったバンドパスフィルタで除去
- 日照時間の季節要因は月日ごとの日照時間の全期間での最大値から理論値を近似して除去
-- ⇒　 アップデート版では、緯度から求められる理論値からのずれと温度の相関を次ごとに求め、季節要因を除去
- 上記処理を行った温度に対して、前項日照時間、および降雨量、平均風速との各々相関係数を求めて依存成分を除去
