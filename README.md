# 📍 Pythonプログラミング演習

Python 学習で作成したプログラムです。
---
## プログラム一覧
| ソース名 | 概要 |
|---|---|
| main.py| eStatのAPIを使って都市コードをキーにその都市の人口を求めるプログラム |
| juki.py | eStatから住民基本台帳のExcelデータファイルをダウンロードするプログラム |
| pytest.py | ダイアログで入力した市町村の5年分の人口推移を、juki.pyで落としたExcelファイルから抽出してグラフ化 |

## インストールしたライブラリ
| ライブラリ名 | 目的 |
|---|---|
| pandas| データ操作 |
| openpyxl | Excelデータアクセス |
| matplotlib | グラフ描画 |
